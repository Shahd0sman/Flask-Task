# Flask Game Project

A small Flask web application with login/signup forms and a moving cat on the homepage.

## Features
- Login and Signup forms with SQLite database
- Flash messages for login errors
- Animated robot on the home page
- 
## Installation

1. Clone the repository:
https://github.com/Shahd0sman/flask-game-project.git
2. Go to project directory:
cd "flask-game-project"
3. Create virtual environment (optional but recommended):
python -m venv venv
venv\Scripts\activate
4. Install requirements:
pip install -r requirements.txt
5. Run the app:
python home.py
